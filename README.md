# The Wild Oasis

A cabin reservation app built with Next.js, Tailwind CSS, and TypeScript.

## Live Demo

[https://the-wild-oasis-73.vercel.app](https://the-wild-oasis-73.vercel.app)

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- NextAuth
- Beautiful icons with Heroicons

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
